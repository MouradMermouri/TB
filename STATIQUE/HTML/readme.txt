Exercice d'application

	Vous devez reproduire la page HTML suivante qui constitue la page d'accueil du site de l'entreprise Jarditou.

		1 - Vous veillerez bien sûr à utiliser les balises HTML 5 telles que <header>, footer>, <nav>, <section>, <article> etc.

		2 - Copiez le fichier HTML de cette première page et renommez la copie tableau.html.
	  		2.1 - Dans cette nouvelle page, insérez le tableau HTML de la phase 7 en remplacement du texte d'accueil.

		3 - Renouvellez l'opération pour insérer le formulaire de la phase 8 dans une page nommée contact.html.
			3.1 - Adaptez les liens de la barre de navigation pour permettre de naviguer entre les 3 pages du site.

		4 - Après avoir réalisé les pages, faites CTRL+U dans Chrome ou Firefox pour afficher le code source, 
			copiez le (CTRL+A et CTRL+C) puis rendez-vous sur le validateur HTML du W3C : 
			choisir l'onglet Validate by Direct Input et copiez-y votre code. 
			Cliquez sur le bouton Check et vérifiez qu'il n'y ait pas d'erreurs.
